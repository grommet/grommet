### Install

  You can install Grommet using either of the methods below.

  For Bower users:
  ```
    $ bower install grommet
  ```

  For NPM users:
  ```
    $ npm install -g grommet
  ```

  Detailed instructions are on the [Get Started](http://grommet.io/docs/documentation/get-started) page.
